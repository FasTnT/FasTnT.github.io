FasT&T - EPCIS 1.2 endpoints

Implemented functionalities

FasT&T is a full implementation of the EPCIS 1.2 specification, and include several HTTP endpoints to allow managing the data.

  • Capture
    • Events
    • Capture Master Data (CBV)
  • Queries:
    • GetVendorVersion
    • GetStandardVersion
    • GetQueryNames
    • GetSubsciptionIDs
    • Poll:
      • SimpleEventQuery
      • SimpleMasterDataQuery
  • Query Callback:
    • CallbackResults
    • CallbackQueryTooLargeException
    • CallbackImplementationException
  • Subscriptions:
    • Subscribe to an EPCIS request
    • Unsubscribe from EPCIS repository
    • Trigger subscriptions that register to specific trigger name
    • Execute subscription based on schedule
