FasT&T - EPCIS repository

Introduction

FasTnT EPCIS is an implementation of the GS1 EPCIS 1.2 specification in .NET.

Setup

Prerequisites:

  • SQL Server 2016 or +
  • .NET 5 SDK

Steps:

  1. Download the source code or clone the repository on GitHub;
  2. Start the repository with the command $ dotnet run -p FasTnT.Host\FasTnT.Host.csproj --urls "http://localhost:5102/" --connectionStrings:FasTnT.Database "{your connectionstring}"

That's it! The database should be created automatically and you should be able to use the endpoints to capture or query events.

Sandbox

FasT&T EPCIS repository is deployed on azure and accessible to the public for testing purpose. You can have more details on the sandbox page.

The repository is hosted on Microsoft Azure platform.
