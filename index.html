Introduction

FasTnT EPCIS is an implementation of the GS1 EPCIS 2.0 and 1.2 specification in .NET.

Setup

Prerequisites:

  • SQL Server 2016 or +
  • .NET 6 SDK

Steps:

  1. Download the source code or clone the repository on GitHub;
  2. Start the repository with the command $ dotnet run -p src\FasTnT.Host\FasTnT.Host.csproj --urls "http://localhost:5102/" --connectionStrings:FasTnT.Database "{your connectionstring}"

That's it! The database should be created automatically and you should be able to use the endpoints to capture or query events.
