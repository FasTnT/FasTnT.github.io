FasT&T - EPCIS 2.0 endpoints

Implemented functionalities

FasT&T is an implementation of the EPCIS 2.0 specification, and include several HTTP endpoints to allow managing the data.

  • Capture
    • Capture list of Events
    • Capture single event
  • Queries:
    • List events (+ filters)
    • Create a named query
    • Execute a named query
  • Subscriptions:
    • Subscribe to a named query via webhooks
  • Discovery:
    • EventType discovery endpoint
    • EPCs discovery endpoint
    • Business Steps discovery endpoint
    • Business Locations discovery endpoint
    • Read Points discovery endpoint
    • Dispositions discovery endpoint

All the events captured using the v2.0 endpoints are compatible with the v1.2 endpoints and vice-versa.

Likely, all the events captured can be queried in either XML of JSON format independently of the language used to capture them.
